About This Project

This project implements CycleGAN for unpaired image-to-image translation using cycle-consistent adversarial networks. The architecture uses two generators and two discriminators with cycle consistency loss to learn mappings between two image domains.

It can perform style transfer, object transfiguration, season transfer, and photo enhancement without paired data.
